## Installation

Needed: Python 3.5

Preferably under `virtualenv`:

`pip install pip-tools` (once)

`pip-sync requirements*.txt` (keeping the PyPI dependencies up-to-date)


## Testing

`./test.sh` (also generates a coverage)
